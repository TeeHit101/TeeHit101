### 🌟 Utvalda Projekt

#### 🖥️ [k8s-infrastructure](https://github.com/TeeHit101/k8s-infrastructure)
Ett komplett, produktionslikt hemlabb byggt på **k3s** och hanterat deklarativt med **Flux Operator** (GitOps).
* **IaC & GitOps**: Flux Operator, Kustomize.
* **Storage & Network**: Longhorn CSI (distribuerad blocklagring), MetalLB L2 LoadBalancer, Nginx Ingress.
* **Säkerhet**: Kyverno-säkerhetspolicyer, NetworkPolicies (Namespace-isolering), Cert-Manager (Let's Encrypt).
* **Observability**: Prometheus & Grafana för metrik, Loki & Promtail för logg-aggregering.
* **CI/CD**: GitHub Actions som provbygger med Kustomize och skannar efter säkerhetshål med Trivy.
